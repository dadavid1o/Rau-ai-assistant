# RAU AI Assistant

RAU AI Assistant — это консольное приложение для поиска информации по дисциплинам магистерских программ RAU.  
Проект помогает быстро находить данные о курсах из учебных планов и УМКД без ручного просмотра большого количества PDF-файлов.

## Problem

Информация о дисциплинах магистратуры хранится в разных источниках: в учебных планах, УМКД и вспомогательных таблицах.  
Из-за этого поиск данных вручную занимает много времени: нужно открывать документы, искать код курса, название, семестр, кредиты, описание, результаты обучения и компетенции.

## Goal

Цель проекта — создать локального консольного ассистента, который позволяет быстро получать информацию по дисциплинам магистратуры из одной базы данных.

Пользователь должен иметь возможность:
- найти дисциплину по коду или названию;
- узнать семестр и количество кредитов;
- посмотреть описание курса;
- получить результаты обучения и компетенции;
- видеть источник, из которого были взяты данные.

## Solution Approach

Проект устроен следующим образом:

1. Из учебных планов формируется базовый список дисциплин в формате CSV.
2. Эти данные загружаются в SQLite-базу.
3. После этого база дополняется данными из УМКД:
   - description
   - learning_outcomes
   - competencies
   - source
4. Через CLI пользователь вводит запрос, а приложение ищет нужную дисциплину в базе.
5. При наличии OpenAI API система может формировать более удобный текстовый ответ на основе найденных данных.
6. Для воспроизводимости обновления по УМКД сохраняются в SQL-файлах.

## Current Result

На текущем этапе проект уже поддерживает две магистерские программы:
- **MatMod**
- **MLDS**

Сейчас реализовано:
- хранение данных о дисциплинах в SQLite;
- импорт базовых данных из CSV;
- поэтапное заполнение базы на основе УМКД;
- поиск курсов через CLI;
- сохранение SQL-обновлений для повторного применения к базе.

Подготовлены отдельные SQL-файлы:
- `data/sql/matmod_umkd_updates_clean.sql`
- `data/sql/mlds_umkd_updates_clean.sql`

Они позволяют восстановить содержимое базы после пересоздания без ручного повторного ввода.

## Current Data Status

### MatMod
Для программы MatMod заполнены все дисциплины, по которым удалось найти подтвержденные материалы.  
Практики, ВКР и дисциплины без надежных исходных данных оставлены пустыми без выдумывания информации.

### MLDS
Для программы MLDS заполнены все обычные дисциплины, по которым удалось найти УМКД.  
Практики, ВКР и отдельные дисциплины без надежных материалов оставлены незаполненными сознательно.

## Project Structure

- `src/` — основной код приложения  
  - `app.py` — CLI-интерфейс  
  - `db.py` — работа с SQLite  
  - `import_csv.py` — импорт данных из CSV  
  - `llm.py` — работа с OpenAI API  

- `data/` — данные проекта  
  - `matmod_skeleton.csv` — базовый список дисциплин MatMod  
  - `mlds_skeleton.csv` — базовый список дисциплин MLDS  
  - `sql/` — SQL-обновления для заполнения базы из УМКД  

- `ucheb_planer/` — PDF-файлы учебных планов  
- `courses.db` — основная SQLite-база данных  
- `README.md` — описание проекта  
- `tools_extract_mlds.py` — вспомогательный скрипт для работы с данными MLDS
